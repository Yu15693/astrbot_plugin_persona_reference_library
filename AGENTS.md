# AGENTS.md

## 作用范围

本说明文件适用于当前插件仓库根目录及其子目录。

## 约定

1. 代码注释与测试 docstring 首选中文，标识符、API 字段名、协议级字面量保持不变。
2. 测试默认保持可重复、可稳定执行；外部真实调用必须显式开启。
3. 在“整体验收通过”前，不要频繁运行测试。开发阶段以实现为主，仅做最小必要检查；到验收阶段再进行完整测试。
4. `src/utils` 与 `src/tools` 的分层约定：
   - `src/utils`：通用工具函数，尽量不依赖宿主实现细节，可在纯插件业务逻辑中复用。
   - `src/tools`：与宿主（AstrBot）强相关的适配工具，处理事件/消息等平台集成逻辑。
   - 两者虽然都可能是“工具函数”，但耦合边界不同，需按职责放置，避免混用。
